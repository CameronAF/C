This is a program to create a team roster of units for arenas events in the game of Fantasica.
Fantasica is a massively multiplayer online game (MMO) for Android or Apple.
5 units are hardwired into the program.
This program is NOT case sensitive
	
	WHEN ADDING AN ENTRY
-NOT case sensitive
-space sensitive(use underscore[_])
1) NAME: If the name contains a space, a underscore must be used instead of the space 
	[ex: Goddess_Danu]
2) COST: integers of 10, 15, 20, 25, or 30
3) SPEED: character of S, A, B, C, D, or E(fast-slow)
4) ATTACK: Land, Air and Sea attack will be integers greater than or equal to 0
5) AVARAGE ATTACK: calculated by program
6) SKILL: either None, Poison, Slow, Knockback, Area_Attack, Attack_Boost(M), or Attack_Boost(F)
7) VALUE: a decimal in the hundredths place[ex: 0.25] between 1 and 0(not including 0)

	DELETE A ENTRY
-Choose a name form the roster to be deleted
-NOT case sensitive
