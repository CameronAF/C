This is a Number Guessing game Console Application 
 
HOW TO USE:
User guess a number between 1 to 100 with 6 attempts
 1) User inputs a number from 1 to 100
     - User is told if number is too big or to small
 2) repeat step 1 5 more times
     - after 6 incorrect guesses, the user loses.
	 
ABOUT THE PROGRAM: 
A function is used to create a file of random numbers using a random number generator.
These number are the numbers the game uses.

Simple program that uses functions, printf, scanf and simple mathmatical calculations
