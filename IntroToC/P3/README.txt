This is a simple Console Application

HOW TO USE:
 1) User enters the price of an item.
 3) Sales tax is calculated for the item:
	 - 7% if the item costs less than $200.
	 - 5% if the item costs $200 or more.
 4) Original item price and sales tax, ant total price is displayed.

ABOUT THE PROGRAM:
This project Uses user-defined functions to calculates different sales tax based on an item price
