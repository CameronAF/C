This is a simple Console Application
 
This project is a ATM simulator
User will be able to make deposits, make withdrawals, or view ballance for 3 seporate accounts

1) User gets prompt to either Deposit, Withdraw, or Check Balance
2) User gets prompt to use Checking, Savings, or Credit
	- requested acction gets executed
3) User is asked if they with to make another transaction
	- if 'Yes', repeat from step 1
