This is a simple Console Application that simulates an ATM machine

HOW TO USE:
 1) User chooses to Deposit, Withdraw, or Check Balance
 2) User chooses to use Checking, Savings, or Credit
 3) If Deposit or Withdraw, User inputs an amount
 	 - Action gets executed
 4) User is asked if they wish to make another transaction
	 - if 'Yes', repeat from step 1

ABOUT THE PROGRAM:
User will be able to make deposits, make withdrawals, or view balance for 3 separate accounts.

Simple program that uses functions, printf, scanf and simple mathmatical calculations
